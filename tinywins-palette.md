## Usage Guidelines

### Primary Actions & CTAs
Use `--accent` (#BF3143) for:
- Primary buttons
- Important links
- Call-to-action elements

### Positive Status & Growth
Use sage greens for:
- "Live" badges
- Success states
- Growth indicators
- Completion states

### Text Hierarchy
- **Headlines**: `--text` (#1e293b)
- **Body/Secondary**: `--muted` (#64748b)
- **Muted/Tertiary**: Reduce opacity of `--text`

### Surfaces & Cards
- **Background**: `--card` (#ffffff)
- **Hover**: `--soft` (#f0f4f3)
- **Border**: `--border` (#e2e8f0)

### Illustrations
Stick to the plant palette (#a6c5a3, #cbdfa5, #7596b3, #324252) for:
- Decorative SVGs
- Icons
- Visual consistency

## Color Combinations

### Success Pattern (Green)
```css
background: #e8f4ed;  /* --greenBg */
border: #a6c5a3;      /* --greenBorder */
color: #324252;       /* --greenText */
```

### Primary CTA Pattern (Red)
```css
background: #BF3143;              /* --accent */
color: #ffffff;
hover-background: #a52a3a;        /* Darker shade */
```

### Card Pattern
```css
background: #ffffff;              /* --card */
border: #e2e8f0;                  /* --border */
hover-background: #f0f4f3;        /* --soft */
```

### Day Mode Pill Patterns

background: #FEF2F2;              /* Light pink */

## Accessibility Notes

- **Text on white background**: Use `--text` (#1e293b) for AAA contrast
- **Muted text**: `--muted` (#64748b) provides AA contrast on white
- **Accent red**: Ensure sufficient contrast when used with text
- **Green pills**: Dark text (#324252) on light background (#e8f4ed) ensures readability
